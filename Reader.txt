## ğŸš€ Quick Start

### Prerequisites

- Node.js 16+
- npm 8+

### Installation

1. **Clone or download the project files**
1. **Install dependencies:**
   
   ```bash
   npm install
   ```
1. **Build the TypeScript code:**
   
   ```bash
   npm run build
   ```
1. **Start the development server:**
   
   ```bash
   npm run dev
   ```
1. **Open your browser** to `http://localhost:5173`
